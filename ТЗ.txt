Цель приложения:

Разработать backend-приложение на Django с использованием Django REST Framework для:
-регистрации пользователей на игры;
-хранения профилей игроков;
-управления данными об играх;
-администрирования доступа (бан/допуск);
-отображения календаря игр;
-работы с изображениями и Telegram-ссылками;
-контейнеризации (Docker).

Используемые технологии:
Python 3.10+
Django 4+
Django REST Framework
Pillow
Docker/Docker-compose
PostgreSQL 

Модели 
1. UserProfile (игрок)
2. Event (событие)
3. EventRegistration (участие в событии)
4. Calendar

API-функционал

Игроки - Регистрация, просмотр профиля, обновление
Игры - Получение списка игр, просмотр, регистрация
Календарь - Получение списка всех игр
Админ - Бан/разбан, просмотр регистраций

Задачи для разработки:

- Создать проект и приложение Django.
- Подключить PostgreSQL и Pillow.
- Настроить Dockerfile и docker-compose.yml.
- Создать модели UserProfile, Game, GameRegistration, Calendar.
- Настроить админку для удобного управления.

API-интерфейсы
Создать сериализаторы для всех моделей.
Реализовать CRUD API для Game (просмотр, создание, изменение, удаление).
Реализовать API для регистрации пользователя на игру (/games/<id>/register/).
Реализовать API для отображения календаря всех игр.
Добавить эндпоинт для получения профиля игрока.
Добавить возможность фильтрации игр по дате.

Дополнительные функции
-Реализовать API для блокировки игрока (бан).
-Реализовать автоопределение команды как уникального значения.
